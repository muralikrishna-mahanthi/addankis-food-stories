# AFSPL — Corporate FMCG Brand Website

Live Site: [afsplcom.com](https://afsplcom.com)
Platform: Shopify (Custom Theme Development)

A complete corporate + eCommerce website built on Shopify for Addanki's FMCG & SPICES PVT LTD (AFSPL) —
an emerging Indian FMCG company operating across food, lifestyle, and distribution verticals.

---

# What I Built

This wasn't a standard Shopify store — it's a full corporate web presence with eCommerce integrated,
built from scratch with heavy customization on Shopify's theme architecture with complete payment integration to function the store dynamically live.

---

# Design & Animations

- Animated hero section with rotating text (Emerging Indian FMCG Company → Brand Builder → Distribution Leader...)
- Infinite scrolling logo marquee for distribution partners
- Auto-sliding product image carousel showcasing all brand products
- Scroll-triggered section reveals and smooth transitions throughout
- Fullscreen background video section with marquee text overlay ("LIFESTYLE. FOOD. SMILES.")
- Clean corporate layout balancing brand storytelling with product commerce

---

# 4 Child Brands Integrated

Each brand has its own dedicated page, identity, and product collection:

| Brand | Category |
| **HAMSAA** | Food / FMCG Products |
| **JINAA** | Lifestyle / Wellness |
| **STRATICA OS** | Health & Functional |
| **FRUROOT** | Fruit-Based Products |

All 4 brands are seamlessly housed under the AFSPL parent umbrella with individual brand pages,
navigation, and linked product collections.

# Shopify Features Implemented

- Custom Shopify theme with section-based architecture

- Cart drawer with promo logic (BUY 2 GET 1 FREE)
- Customer login & checkout integration
- Retailer / Distributor Inquiry page
